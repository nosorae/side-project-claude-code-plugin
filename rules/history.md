# 히스토리 기록

유의미한 작업 완료 시 대화 기록을 남긴다.

- 기록 위치: `docs/dialogs/{제목}-{날짜시간}.md` (예: `docs/dialogs/로그인-기능-구현-20260303-1430.md`)
- 기록 내용: 사용자 명령 원본, 과정 요약 (주요 결정 포인트 포함), 최종 결과
- 기록 시점: 기능 구현, 아키텍처 결정, 중요한 버그 수정 등 유의미한 작업 완료 시
- 기록 후 즉시 커밋+푸시하여 히스토리 유실 방지
- 사소한 수정(오타, 포맷팅 등)은 기록하지 않는다

## 강제 메커니즘

`Stop` hook(`hooks/enforce-dialog.sh`)이 세션 종료 시 자동 검사한다:
- 최근 4시간 내 작업 커밋이 있는데 `docs/dialogs/` 커밋이 없으면 종료를 차단한다
- 대화 기록을 작성하고 커밋+푸시해야 세션 종료가 가능하다
