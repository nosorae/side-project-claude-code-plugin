# App Plan 예시

## Example 1: 운동 루틴 앱

**User says:**
```
앱 기획해줘. 퇴근 후 운동하는 직장인들을 위한 루틴 공유 앱이야.
```

**실행 흐름:**

1. **Step 1** — 에이전트 A: 시장 5,000억 성장 / B: Home Workout 500만DL이지만 한국어 루틴 부족 / C: 레드오션 리스크 → opus 판정 ⭐⭐⭐⭐☆
2. **Step 2** — D: '오늘의 루틴 체크' 핵심 / E: 프리미엄 월 4,900원 / F: 체크리스트만이면 1주 → opus 결정: 체크리스트만, 타이머/소셜은 v2
3. **Step 3** — 화면 4개, Next.js + Supabase (PWA)
4. **Step 4** — 핵심 플로우: 앱 열기 → 루틴 생성 → 체크 완료, 3초 룰 통과
5. **Step 5** — `docs/ssot/prd/2026-02-08-운동루틴앱-기획서.md` 저장

**Result:** 중간 확인 없이 연속 실행 + 최종 결과 한 번에 리뷰

## Example 2: 중간에 방향 전환

경쟁이 너무 심한 것 같은데... 다른 아이디어로 바꾸고 싶어.
